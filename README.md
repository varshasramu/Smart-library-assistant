# Smart Library Assistant - Voice-Powered Web App

A lightweight, voice-powered web application for library management with minimal compute power requirements.

## Features

### For Users:
- **Voice-Powered Search**: Use voice commands to find books
- **Book Search**: Search by title, author, or genre
- **Personalized Recommendations**: Get book suggestions based on genre
- **Synopsis on Demand**: Read book summaries with text-to-speech capability
- **Responsive Design**: Works on mobile and desktop devices

### For Admins:
- **Add New Books**: Simple form to expand the library catalog
- **Library Statistics**: View counts of books, authors, and genres
- **User/Admin Mode Toggle**: Switch between user and admin interfaces

## Technical Features
- **Minimal Compute Power**: Lightweight JavaScript with no heavy frameworks
- **Voice Interaction**: Simulated Snips-NLU-like functionality
- **Text-to-Speech**: Browser-based speech synthesis
- **Local Storage Simulation**: In-memory data storage (can be extended to backend)
